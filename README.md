# Grovemoor

Small test project with Vue and GitHub Pages

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```

## Push to GH Pages

```sh
git subtree push --prefix dist origin gh-pages
```
